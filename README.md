# This is Green Rain
## Clone git and run

``` bash
$ git clone https://github.com/ductnn/Green-Rain
$ cd Green-Rain
$ npm install
$ npm run dev
```
**Start** and **Play** :))) 
You can see demo in [here](https://green-rain.herokuapp.com/)
